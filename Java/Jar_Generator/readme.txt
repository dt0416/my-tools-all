== Method 1:Use jar command ==
1. copy classes which compiled into classes folder
2. setup mymanifest.txt Main-Class(if gen Executable Jar)
3. run 01 bat
4. output jar to dst folder

== Method 2:Use ANT command ==
1. with Method 1 step 1
2. setup build.xml Main-Class(if gen Executable Jar) 
3. run 02 bat
4. output jar to dst folder
